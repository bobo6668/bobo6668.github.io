2. 常用名词

  • App：应用（Application）的缩写
  • 手机内存 包含两个：一个是运行内存(RAM)，一个是机身内存(ROM)。参考 手机内存RAM、ROM简介
    • 运行内存(RAM)：储存手机操作系统（如安卓系统）、应用程序运行过程中产生的临时数据。
      • RAM(Random Access Memory) 相当于电脑的内存：内存越大，我们可以读写的动态数据就越多，这样手机的反应速度就会越快。换句话说，RAM的大小决定了手机可同时运行多少个程序而不卡顿。
      • 现在的很多智能系统手机都具备后台运行的功能，所有在后台运行的程序其实都是被存在了手机的RAM上，需要使用的时候才会被调用出来，因此只要手机的RAM越大，可同时运行的程序就越多，一旦运行的程序超过了手机内存的容量，那手机就会出现异常，这样的异常一般就是卡死或是死机。
      • 因此，为了让手机运行流畅（节省内存），可以控制手机后台运行程序的数量。控制后台程序的数量除了要手动”关进程”之外，还要防止一些程序自动在后台自动开启
        • 点击跳转到 华为手机怎么关闭后台运行程序
        • 点击跳转到 华为手机怎么开启/禁止后台自动运行应用
    • 机身内存(ROM)：储存手机操作系统（如安卓系统）、应用程序、用户数据（照片、音乐）等静态数据。
      • ROM(Read Only Memory) 相当于电脑的硬盘，比如手机SD卡(Secure Digital Memory Card)就属于ROM范畴。硬盘越大，可以存放的静态数据就越多。换句话说，硬盘越大，手机就可以储存更多的照片、音乐、视频等，可以安装更多的程序。
      • 节省ROM空间跟用户的使用习惯密切相关。在手机内定期清理下载的安装包和删除不常用的软件，不仅有利于节省ROM，也会让手机反应更迅速（ROM储存太满会拖慢手机整体反应速度）。
        • 点击跳转到 3.1.2. 清理垃圾
  • 刷机相关
    • ROM：刷机包
    • 白苹果：指iOS设备出现软硬件故障，卡在启动画面上的现象，一般多出现于越狱后插件冲突或系统bug。
    • root：root用户是系统中唯一的超级管理员，它具有等同于操作系统的权限。
      • 优点：一些需要root权限的应用，譬如广告阻挡，是需要root权限的。
      • 缺点：问题在于root比windows的系统管理员的能力更大，足以把整个系统的大部分文件删掉，导致系统完全毁坏，不能再次使用。所以，用root进行不当的操作是相当危险的，轻微的可以死机，严重的甚至不能开机。因此，除非确实需要，一般情况下都不推荐使用root。
    • 一键root：是一些软件的功能，可以一键获取超级用户权限
    • Recovery：
      • Recovery是Android手机备份功能，指的是一种可以对安卓机内部的数据或系统进行修改的模式（类似于windows PE或DOS）。在这个模式下我们可以，对已有的系统进行备份或升级，也可以在此恢复出厂设置。
      • 刷入第三方的Recovery，将获得更多的功能，并且可以刷入第三方rom，官方自带则不行。

5.3. WIFI万能钥匙（流氓软件，不建议使用）

wifi万能钥匙所谓的“破解”过程，其实，是对你的手机里面保存的wifi密码等信息的窃取，获取到你所需要连接的wifi的mac然后遍历服务器，找到匹配的然后下发密码测试。

摘自：如何评价应用「WiFi 万能钥匙」？ - 知乎
https://www.zhihu.com/question/23874535

​ 也就是强行分享，你下载了，就已经先泄露了自己的手机上已经保存的wifi密码。你能够用到的wifi，也都是别人泄露的。这是伤敌一千自损八百的做法。

​ 因此，建议不安装这个流氓App。

​ 此外，也不需要安装360杀毒等挂羊头卖狗肉的流氓软件。
